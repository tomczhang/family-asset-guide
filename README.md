# 家庭资产应急手册

一款完全离线运行的家庭资产应急手册生成工具。帮助你整理家庭资产信息、密码凭证、紧急操作流程，一键导出为 AES-256 加密 PDF，确保关键时刻家人能找到并处置所有资产。

**在线使用：https://tomczhang.github.io/family-asset-guide/**

## 核心特性

- **完全离线** — 所有数据仅存在于浏览器内存，不经过任何服务器，零泄露风险
- **军事级加密** — 导出为 AES-256 加密 PDF，密码是唯一的解锁方式
- **PDF 即草稿** — 加密 PDF 内嵌完整草稿数据，导入 PDF 输入密码即可继续编辑，一个文件既是成品也是存档
- **三步极简** — 选机构 → 填账号 / 资产 / 密码存放地 → 一键生成
- **模块化** — 资产清单、密码指引、紧急响应流程、自定义区四大模块，可按需删除或保留
- **草稿导入导出** — 支持 JSON 草稿和加密 PDF 两种格式导入，随时中断、随时继续

## 功能模块

### 资产清单
录入各类资产信息：美股、港股、A 股、银行存款、保险、不动产、加密货币等。内置常见金融机构数据（券商网址、客服电话、APP 下载链接），选择机构后自动填充。

### 密码指引
记录密码存放位置（保险柜 / 1Password / 密封信封等）、2FA 验证方式及恢复指引，关联到具体资产。继承人通过这里找到登录所需的全部凭证。

### 紧急响应流程
预置五阶段标准操作流程（紧急响应 → 境外资产锁定 → 境内资产处置 → 保险理赔 → 长期事项），可根据家庭实际情况自由修改。

### 自定义区
添加任意内容：遗嘱提示、家族信托说明、律师联系方式、重要备忘等。

## 技术栈

- **前端**：React 18 + TypeScript + Vite
- **PDF 生成**：@cantoo/pdf-lib（支持中文字体嵌入 + AES-256 加密）
- **打包**：vite-plugin-singlefile（构建为单个 HTML 文件，可离线使用）
- **部署**：GitHub Pages（自动 CI/CD）

## 本地开发

```bash
cd inheritance
npm install
npm run dev
```

浏览器访问 `http://localhost:5174`。

## 构建

```bash
cd inheritance
npm run build
```

产物在 `inheritance/dist/index.html`，是一个自包含的单 HTML 文件，可直接双击打开使用。

## 安全说明

- 所有数据仅存在于当前浏览器页面，关闭即丢失，请及时导出
- JSON 草稿文件**未加密**，请存储在加密磁盘或安全位置
- PDF 使用 AES-256 加密，密码是唯一的解锁方式，请务必记牢
- 需要 Chrome 103+ 或 Edge 浏览器（依赖 `queryLocalFonts` API 读取系统中文字体）
